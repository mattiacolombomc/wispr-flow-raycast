# Wispr Flow for Raycast

Control Wispr Flow voice-to-text directly from Raycast. Start and stop voice recording without leaving your keyboard.

## Features

- **Start Recording** - Begin voice dictation instantly
- **Stop Recording** - End recording with a single command
- **Open App** - Quick access to Wispr Flow app
- **Installation Check** - Automatic detection with download link if not installed

## Commands

| Command | Description |
|---------|-------------|
| Wispr Flow | Main menu with all actions |
| Start Recording | Begin voice dictation (no-view) |
| Stop Recording | End voice dictation (no-view) |
| Open Wispr Flow | Open the app (no-view) |

## Requirements

- macOS 12.0+
- [Wispr Flow](https://wisprflow.ai) installed
